# Twitter_unfollow_tool

TypeScriptで作成したChrome拡張（Manifest V3）です。  
「自分はフォローしているが、相手は自分をフォローしていない」アカウント候補を抽出し、上限件数まで解除します。

## 構成

- `manifest.json`
- `popup.html`
- `src/background.ts`
- `src/content.ts`
- `src/popup.ts`

## セットアップ

```bash
npm install
npm run build
```

ビルド後、`chrome://extensions` で

1. 右上「デベロッパーモード」をON
2. 「パッケージ化されていない拡張機能を読み込む」
3. このフォルダを選択

## 使い方

1. `x.com` にログイン済みの状態で、任意のXタブを開く
2. 拡張アイコンを押して、上限件数を入力
3. `候補件数を取得` を押す
4. `片思いフォロワーを解除しますか？件数○○件(上限○○)` が表示される
5. `この内容で解除開始` を押す
6. 解除中は `x/y` の進捗表示、必要なら `停止` を押す
7. 終了後に `完了しました` または `停止しました` が表示される

## 試運転のおすすめ

1. 最初は上限 `5` で実行
2. 問題なければ `20`、`50` と段階的に増やす
3. 実行中に `停止` が効くか毎回確認する

## 注意

- Xの画面DOM変更で動かなくなる場合があります（その際はセレクタ調整が必要）。
- 自動操作はアカウント制限リスクがあるため、少量運用で様子を見るのが安全です。
